🌍 Geospatial Research 
Welcome to Geospatial Research, a collection of spatial analytics projects exploring geographic patterns, spatial statistics, and location based insights. This repository is maintained by Muluwerk Derebe, a statistician and data analyst with extensive experience in statistical modeling, research methodology, and modern geospatial tools. This space brings together reproducible workflows, geospatial datasets, and analytical scripts designed to support academic research, applied analytics, and portfolio ready demonstrations.

🎯 Purpose of This Repository This repository serves as a central hub for:
    •	Spatial data analysis and visualization
    •	Geostatistical modeling and spatial inference
    •	Exploratory mapping and geographic pattern detection
    •	Applied geospatial research for public health, environment, and socio economic studies
    •	Reproducible workflows using Python, R, and GIS tools Each project emphasizes methodological clarity, clean code, and meaningful interpretation of spatial patterns. 
  
🧭 Research Themes The repository includes studies and examples related to:
  •	Spatial autocorrelation (Moran’s I, Geary’s C)
  •	Hotspot analysis (Getis Ord Gi*)
  •	Spatial regression (SAR, CAR, GWR)
  •	Interpolation techniques (Kriging, IDW, Spline)
  •	Raster and vector data processing
  •	Geospatial visualization and thematic mapping
  •	Remote sensing preprocessing and analysis
  •	Public health and environmental geospatial analytics
